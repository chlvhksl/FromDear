# 🚀 FromDear 배포 매뉴얼

작업을 마치고 사이트를 업데이트할 때, 이 순서대로 따라 하세요!

## 1단계: 작성한 코드 저장하기 (Git)
내가 수정한 내용을 안전하게 깃허브 창고(`main`)에 저장하는 과정입니다.

터미널에 다음 세 줄을 순서대로 입력하세요:

```bash
# 1. 변경된 파일 모두 선택
git add .

# 2. 변경 내용 메모 남기기 (따옴표 안에 내용을 적으세요)
git commit -m "작업한 내용 입력"

# 3. 깃허브로 쏘아 올리기
git push origin main
```

> **확인:** 여기까지 하면 GitHub의 **Code** 탭이 최신으로 바뀝니다.

---

## 2단계: 사이트 업데이트하기 (Deploy)
저장된 코드로 실제 접속 가능한 웹사이트(`gh-pages`)를 만드는 과정입니다.

```bash
npm run deploy
```

> **확인:** 
> * 터미널에서 `Published`라고 뜨면 성공!
> * 약 2분 뒤에 `https://fromdear.github.io` 에 들어가면 바뀐 내용이 보입니다.

---

## 💡 요약
작업이 끝나면 항상 **"저장(Push) 하고 배포(Deploy) 한다"** 
두 단계를 기억해주세요!
